📌 Introduction

This project is a beginner-to-intermediate level orbital simulation tool designed to model and visualize rocket trajectories in a space-centric (global) frame of reference. Instead of a ground-based perspective, this simulation treats celestial bodies like Earth, Moon, and Sun as moving entities within space, allowing users to observe orbital mechanics more realistically.

The goal of this project is to help understand fundamental physics concepts such as gravitational interaction, orbital motion, and trajectory prediction while also serving as a foundation for more advanced simulation systems in the future.


---

⚙️ Features

🌍 Space-perspective simulation (not Earth surface view)

🌙 Dynamic motion of celestial bodies (Earth, Moon, etc.)

🚀 Rocket trajectory prediction and visualization

🔁 Toggle between trajectory planner and 3D trajectory view

🧭 Visual orientation aids (compass-like axis system)

📊 Basic physics-based calculations for motion and gravity

🧩 Modular structure for future expansion

🛠️ Installation & Setup

1. Prerequisites

Make sure you have:

Python 3.x installed

pip (Python package manager)


2. Clone the Repository

git clone https://github.com/your-username/rocket-orbital-simulation.git
cd rocket-orbital-simulation

3. Install Dependencies

pip install numpy scipy matplotlib

(Optional: If using virtual environment)

python -m venv venv
source venv/bin/activate   # Linux/Mac
venv\\Scripts\\activate    # Windows

4. Run the Simulation

python main.py
